My Work

Here are projects I am currently working on or have completed. These include projects from Tech Elevator and side projects.

Meal Planner Website

A full-stack website using a PostgreSQL database, Vue.js framework, and Java. The website was designed to be user-friendly and to make meal planning easier for users. Users can add items to their grocery lists from the recipe ingredients and easily add their own recipes to the website to share their favorites!

Movie Website

Developed a full-stack website employing a PostgreSQL database, Vue.js framework, and Java. The website enables seamless exploration of movies tailored to individual preferences. Users can curate a personalized list, facilitating easy reference for future movie selections.

Pong

Developed a classic two-player mini-game in Python utilizing Turtle graphics. Programmed the dynamic movement of a ball, requiring players to skillfully maneuver paddles to prevent the ball from hitting the border behind. Implemented scoring mechanics and showcasing proficiency in Python programming and game development.

Vending Machine Project

Collaborated with a partner to build a vending machine application. The application was designed for users to make purchases and owner to receive sales data. This project was built using Java object-oriented programming.

TEnmo Project

Hangman/Wordle Project

Recreated the classic Hangman and Wordle game with various word list themes. This project was coded using Java on IntelliJ.

Little Lemon Prototype

Used Figma to create a prototype of an app for a Mediterranean Restaurant called Little Lemon. This prototype included a form to fill out reservations. A wire frame was also created for this project.

About Me

My goal is to use the skills I have acquired from Tech Elevator to become a Computer Programmer and to continue to challenge myself.
